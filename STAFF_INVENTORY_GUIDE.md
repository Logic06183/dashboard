## How to Update Inventory (Daily Task)

### Step 1: Access the Stock Entry Page
1. Open the Linden Dashboard (main React app)
2. Look for the **green button** in the sidebar that says **"ðŸ“¦ Daily Stock Entry"**
3. Click it to open the stock entry page

### Step 2: Update Stock Levels
The page shows ALL ingredients used in pizzas, organized in a grid with color-coded cards:

- ðŸ”´ **RED** = Out of Stock (0 items)
- ðŸŸ  **ORANGE** = Low Stock (below threshold)

### Step 3: Three Ways to Update Amounts

**Option 1: Quick Adjust Buttons**
- Use **+1, +10, +100** to add stock
- Use **-1, -10, -100** to reduce stock

**Option 2: Quick Set Buttons**
- Click preset amounts like "0", "100", "500", "1000" to set exact quantities

**Option 3: Direct Input**
- Type the exact amount in the number field

### Step 4: Filter & Search
- Use the **Category dropdown** to view specific types (Cheese, Meat, Vegetables, etc.)
- Use the **Search box** to find ingredients quickly by name

### Step 5: Save Your Changes
1. After updating all stock levels, click the **"Save All Changes"** button at the bottom
2. Wait for the success message
3. Done! The system automatically tracks usage as orders come in

## When to Do This
- **Daily**: At the start of the day after receiving deliveries
- **After major deliveries**: When new stock arrives
- **End of day**: To verify remaining stock

## Tips
- The system shows how much of each ingredient is used per pizza
- Low stock warnings appear automatically when items run low
- The threshold for "low stock" is set to 2x daily usage or minimum 100 units
- All units are shown (g for grams, ml for milliliters, etc.)

## Need Help?
- The page shows current stock value and statistics
- Red/Orange items need immediate attention for ordering
- Contact manager if unsure about any stock levels
